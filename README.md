# 📝 To-Do List Application

A simple, responsive, and aesthetic **To-Do List Application** built using **HTML, CSS, and JavaScript**. This project helps users organize their daily tasks with an easy-to-use and clean interface.

---

## ✨ Features

- ➕ Add new tasks
- 🗑️ Delete completed tasks
- 🎨 Clean and aesthetic UI
- 📱 Responsive design
- ⚡ Built with Vanilla JavaScript (No frameworks)

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)

## 🚀 Getting Started

1. Clone this repository

```bash
git clone https://github.com/your-username/To-Do-List.git
```

2. Open the project folder.

3. Run `index.html` in your browser.
